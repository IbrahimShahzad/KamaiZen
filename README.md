# KamaiZen
A language server for kamailio configuration files


## Features
- [ ] Code completion
    - [ ] Variables
    - [x] Functions
    - [ ] Modules
    - [ ] Keywords
    - [ ] Parameters
    - [x] Headers
- [ ] Code navigation
  - [ ] Go to definition - In progress
  - [ ] Find references - In progress
- [ ] Code formatting
- [ ] Code folding
- [ ] Diagnostics
    - [x] Syntax Errors
    - [x] Invalid statements
    - [x] Unreachable code
    - [ ] Unused variables
    - [ ] Unused functions
    - [ ] Unused modules
    - [ ] Unused parameters
- [ ] Hover
    - [x] Show documentation
    - [ ] Show type
- [ ] Highlight
